# AI App Local Data RAG

This example demonstrates how to build a "defensible AI application" by integrating proprietary data and custom retrieval logic, rather than just being a simple wrapper around an LLM API. It uses a local product database and a simulated LLM to show how an application can add unique value by providing curated context to the AI model. The core value resides in the application's data and logic, making it more robust against changes in underlying AI models.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`
3. Type product-related questions (e.g., "Akıllı Saat X1 fiyatı ne kadar?" or "Kablosuz Kulaklık Y2 özellikleri neler?").
